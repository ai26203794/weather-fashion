# 音声ファイルフォルダ

このフォルダにVOICEVOXで生成した音声ファイルを配置してください。

## 必要なファイル

### 春（spring）
- `spring-sunny.wav` - 春の晴れコーデの説明
- `spring-cloudy.wav` - 春の曇りコーデの説明
- `spring-rainy.wav` - 春の雨コーデの説明
- `spring-snowy.wav` - 春の雪コーデの説明

### 夏（summer）
- `summer-sunny.wav` - 夏の晴れコーデの説明
- `summer-cloudy.wav` - 夏の曇りコーデの説明
- `summer-rainy.wav` - 夏の雨コーデの説明
- `summer-snowy.wav` - 夏の雪コーデの説明

### 秋（autumn）
- `autumn-sunny.wav` - 秋の晴れコーデの説明
- `autumn-cloudy.wav` - 秋の曇りコーデの説明
- `autumn-rainy.wav` - 秋の雨コーデの説明
- `autumn-snowy.wav` - 秋の雪コーデの説明

### 冬（winter）
- `winter-sunny.wav` - 冬の晴れコーデの説明 ✅ 追加済み
- `winter-cloudy.wav` - 冬の曇りコーデの説明
- `winter-rainy.wav` - 冬の雨コーデの説明
- `winter-snowy.wav` - 冬の雪コーデの説明

## VOICEVOXでの音声生成方法

1. VOICEVOXをダウンロード: https://voicevox.hiroshiba.jp/
2. アプリを起動
3. キャラクターを選択（ずんだもん、四国めたんなど）
4. 以下のテキストを入力して音声を生成

## 音声テキスト例

### 春
- **晴れ**: 「春らしい爽やかな天気です！軽やかなトレンチコートや薄手のカーディガンがおすすめ。パステルカラーで季節感を演出しましょう。」
- **曇り**: 「少し肌寒い曇り空です。薄手のジャケットやカーディガンを羽織って、体温調節しやすいレイヤードスタイルがおすすめです。」
- **雨**: 「雨の日はレインコートやウォータープルーフのジャケットを。足元は防水シューズで濡れ対策を忘れずに。」
- **雪**: 「珍しい春の雪です！暖かいコートとブーツで防寒対策を。滑りにくい靴底の靴を選びましょう。」

### 夏
- **晴れ**: 「暑い夏日です！通気性の良い素材のTシャツやワンピースがおすすめ。日焼け対策に帽子やサングラスも忘れずに。」
- **曇り**: 「曇りでも蒸し暑い日。涼しげなリネン素材やコットンの服装で快適に過ごしましょう。」
- **雨**: 「梅雨や夏の急な雨に備えて、折りたたみ傘を持ち歩きましょう。速乾性のある素材がおすすめです。」
- **雪**: 「非常に珍しい天候です。念のため上着を持って外出しましょう。」

### 秋
- **晴れ**: 「過ごしやすい秋晴れです！軽いジャケットやニットカーディガンで季節感を。ブラウンやボルドーなど秋色がおすすめ。」
- **曇り**: 「肌寒くなりそうな曇り空。重ね着で体温調節できるスタイルを。ストールやマフラーをプラスしても◎」
- **雨**: 「秋雨の季節。撥水加工のトレンチコートやブーツがおすすめ。折りたたみ傘もお忘れなく。」
- **雪**: 「早い冬の訪れです。暖かいコートとマフラーで防寒対策を。足元は滑りにくいブーツがおすすめ。」

### 冬
- **晴れ**: 「晴れていても寒い冬。暖かいダウンジャケットやウールコートで防寒を。マフラーと手袋で完璧に。」
- **曇り**: 「底冷えする曇りの日。ヒートテックなどのインナーでしっかり防寒。ニット帽やイヤーマフもおすすめです。」
- **雨**: 「冷たい雨の日。防水性のあるコートと暖かいブーツで。濡れても乾きやすい素材を選びましょう。」
- **雪**: 「本格的な雪の日。ダウンコートやムートンブーツでしっかり防寒。滑りにくい靴底と防水対策が必須です。」

## ファイル形式

- VOICEVOXから直接WAV形式で書き出してそのまま使用できます
- ファイル名は必ず `季節-天気.wav` の形式にしてください（例: `winter-sunny.wav`）

## 注意事項

- 音声ファイルがない場合、ブラウザ内蔵のWeb Speech APIで代替読み上げを行います
- VOICEVOXのキャラクター使用時はクレジット表記が必要な場合があります
